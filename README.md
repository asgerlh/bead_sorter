# Bead sorting machine

## Hardware

RP2040-zero
TCS34725 color sensor module
2x 28BYJ-48 Stepper motors

## Pinout

| Pin | Function |
|-----|----------|
| GP0 | TCS34725 SDA |
| GP1 | TCS34725 SCL |
| GP2 | TCS34725 LED |
| GP4 | Button 1 |
| GP5 | Button 2 |
| GP9 | Motor 1, 1 |
| GP10 | Motor 1, 2 |
| GP11 | Motor 1, 3 |
| GP12 | Motor 1, 4 |
| GP26 | Motor 2, 1 |
| GP27 | Motor 2, 2 |
| GP28 | Motor 2, 3 |
| GP29 | Motor 2, 4 |
